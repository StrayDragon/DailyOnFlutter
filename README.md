# 解决序列化(JSON): 使用codegen技术自动生成序列化对象

- 在项目根目录下:
```bash
#  生成序列化定义
flutter packages pub run build_runner build
```
```bash
# 持续生成
flutter packages pub run build_runner watch
```
- 模板
  - 已调好 `djson` 为 Live Templates 触发键
